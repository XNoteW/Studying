## 本教程的目标：

- 在高层次上理解PyTorch的张量(Tensor)库和神经网络
- 训练一个小型神经网络对图像进行分类
- 本教程假设您对numpy有基本的了解

**注意**： 务必确认您已经安装了 torch 和 torchvision 两个包。
